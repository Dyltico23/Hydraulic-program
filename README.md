# Hydraulic Calculation Software

Software desarrollado en Python para el cálculo hidráulico de sistemas de tuberías y análisis de fluidos.

## Características

- Interfaz gráfica desarrollada con Tkinter
- Cálculo de parámetros hidráulicos
- Visualización gráfica con Matplotlib
- Diseño intuitivo y fácil de usar

## Tecnologías utilizadas

- Python
- Tkinter
- Matplotlib
- NumPy

## Cómo ejecutar

1. Instalar dependencias:

pip install -r requirements.txt

2. Ejecutar el programa:

python main.py
